# 🛠️ AI Agent Skills

## 🧠 Core Reasoning Skills
*   **Chain-of-Thought (CoT)**: Breaking down complex problems into logical steps.
*   **Few-Shot Learning**: Utilizing provided examples to maintain output consistency.
*   **Contextual Awareness**: Maintaining state across multiple agent turns.

## 📝 Business Analysis Skills
*   **INVEST Validation**: Ensuring user stories are Independent, Negotiable, Valuable, Estimable, Small, and Testable.
*   **Gherkin Syntax**: Writing behavior-driven development (BDD) specifications.
*   **Stakeholder Persona Mapping**: Tailoring requirements to specific user roles.

## 💻 Engineering Skills
*   **Modular Design**: Generating code that follows SOLID principles.
*   **API First Design**: Designing robust interfaces before implementation.
*   **Refactoring**: Identifying and fixing technical debt in existing code.

## 🧪 Quality Assurance Skills
*   **Boundary Value Analysis**: Testing the edges of input ranges.
*   **Equivalence Partitioning**: Grouping inputs to minimize test cases while maximizing coverage.
*   **Automated Scripting**: Generating executable test files for CI/CD pipelines.
